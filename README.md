Brainwave - Modern UI/UX website, developed using React.js and Tailwind CSS, exemplifies modern UI/UX principles. Its sleek design, seamless animations, and overall user experience set a high standard, serving as a reference or inspiration for future modern applications or websites in general.
This project is a **personal UI/UX exploration** built to experiment with
modern web design principles and scalable frontend architecture.

The focus of this project is not business logic, but:
- Visual hierarchy
- Layout composition
- Interaction patterns
- Design consistency across sections

It serves as a **showcase of frontend and UI engineering skills**.
